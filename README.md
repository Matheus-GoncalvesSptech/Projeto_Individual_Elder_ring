⚔️ Elder Ring e sua História
"Ergue o Anel Prístino. Torna-te o Lorde Élfico."

Site sobre o jogo Elden Ring, desenvolvido como projeto acadêmico universitário. O site reúne informações sobre os principais chefes, lore, regiões do jogo, uma calculadora de runas e uma história pessoal da minha jornada com o jogo.

🛠️ Tecnologias
HTML5
CSS3 — Flexbox, variáveis de cor, transições
JavaScript — Manipulação do DOM, validações, cálculos
✨ Funcionalidades
🗡️ Galeria de Chefes — Cards com stats, dificuldade, região e drops dos 6 principais chefes
📖 Lore — Resumo da história das Terras Intermédias
🗺️ Regiões — Cards visuais das 4 principais regiões do jogo
🧮 Calculadora de Runas — Calcula o custo exato de runas para subir de nível
🌾 Simulador de Farm — Estima quantas runs e quanto tempo para farmar runas
🔐 Login com validação — Limite de 3 tentativas, credenciais fixas
📝 Cadastro com validação — Verificação de email, senha forte/média/fraca
📖 Minha Jornada — Seção pessoal contando a história real de jogar com meu pai
📚 Contexto Acadêmico
